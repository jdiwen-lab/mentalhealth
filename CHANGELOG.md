# Changelog

本專案的重要版本變更均記錄於此。

## [尚未發行]

### 變更

- 將五種結果完整替換為 1024 × 1536 自製角色圖卡，分享與下載使用原始高畫質 PNG，網頁預覽使用輕量 WebP
- 以深藍、金色、奶油色與五種角色專色重整首頁、題目、結果、說明及隱私區塊
- 新增標準大字與放大字體控制，放大模式下題目選項文字約為 21–23px，主要按鈕及選項具備更大的點選範圍
- 調整手機、平板與桌機版面，讓結果圖卡優先呈現，手機可分享至支援的社群 App，桌機可下載原始 PNG
- 新增自製圖卡風格的社群分享封面、燈塔網站圖示與自架繁體中文字型，避免不同裝置缺字
- 擴充離線快取、圖卡對應測試及靜態資產尺寸與格式驗證

### 驗證

- 七項自動化測試全數通過
- 全部 3,125 種作答組合仍維持唯一、可重現且五種角色各 625 種的平衡結果
- 320px 手機、390 × 844 手機、768px 平板、1440px 與 1920px 桌面瀏覽器 QA 通過
- 已驗證無水平溢位、主控台錯誤、外部請求或 localStorage 留存
- 五張正式 PNG 與使用者提供的原始檔案雜湊完全相同

## [1.0.0] - 2026-07-18

### 新增

- 完整的繁體中文職場韌力角色心理測驗
- 五題單題式自適應作答流程與工作階段續答
- 五種唯一結果及分層同分決勝演算法
- 手機原生圖卡分享與桌機 PNG 下載
- 1200 × 1500 Canvas 結果圖卡
- 鍵盤、螢幕閱讀器、深色模式與降低動態效果支援
- PWA、離線快取、SEO、社群預覽及 GitHub Pages 部署
- 3,125 種答案組合的窮舉測試與 GitHub Actions CI

### 驗證

- 每一種答案組合皆產生唯一且可重現的結果
- 五種角色各 625 種結果
- 桌機與手機瀏覽器 QA 全數通過
