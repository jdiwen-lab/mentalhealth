## [1.3.0] - 2026-07-19

### 變更

- 建置流程改由 `package.json` 自動注入統一資產版本，涵蓋 CSS、所有 JavaScript 模組、manifest、圖示與圖卡
- 偵測既有作答時，將「繼續上次進度」設為主要動作，重新開始前必須經過確認
- 依瀏覽器能力區分「分享圖卡」與「分享測驗連結」，並在圖卡準備期間顯示狀態及鎖定重複操作
- 強化鍵盤焦點、Windows 強制色彩模式及 JavaScript 對減少動態偏好的支援
- 新增 Playwright 完整作答、續答、分享、手機版面與 axe 無障礙回歸測試

### 修正

- Service Worker 只刪除本測驗的新舊快取，避免影響同一 GitHub Pages 網域下的其他專案
- Service Worker 只攔截測驗路徑內請求，並只從目前版本的 Cache 讀取資產
- 重設重新測驗對話框的 `returnValue`，避免取消操作沿用先前確認狀態
- 以共用 Promise 快取高畫質圖卡請求，避免連續點擊造成重複下載

### 驗證

- 10 項單元與靜態檢查全數通過
- 6 項真實 Chromium 流程與 axe WCAG A／AA 自動檢測全數通過
- 全部 3,125 種作答組合仍唯一、可重現且五種角色各 625 種
- 390px 手機放大字體無水平溢位，`npm audit` 未發現已知漏洞
- 題目、角色內容、計分權重與同分決勝程式未變更

## [1.2.0] - 2026-07-19

### 變更

- 依 B 流程精簡首頁，保留測驗重點、開始／續答按鈕、圖卡預覽與單行隱私提醒，移除重複特色、五角色導覽及技術說明
- 精簡作答頁為題數進度、題目、選項與前後操作；作答提示只在第一題出現，續答狀態仍保留於本機工作階段
- 精簡結果頁的重複標題與說明，將其餘四種角色收進預設摺疊的「認識其他角色」區塊

### 修正

- 固定網站使用淺色配色，移除依手機系統深色模式切換的規則，並明確設定根元素與頁面底色
- 更新離線快取版本，避免手機沿用舊版深色樣式與版面
- 為主要樣式與程式加入資產版本識別，避免曾造訪舊版的裝置在首次載入時混用新舊檔案

### 驗證

- 七項自動化測試全數通過
- 全部 3,125 種作答組合仍維持唯一、可重現且五種角色各 625 種的平衡結果
- 正式站首頁、五題作答、結果頁、其他角色摺疊、放大字體與續答流程驗證通過
- 固定淺色方案、版本化資產、無水平溢位及新版網站主控台無錯誤驗證通過

## [1.1.0] - 2026-07-19

### 變更

- 將五種結果完整替換為 1024 × 1536 自製角色圖卡，分享與下載使用原始高畫質 PNG，網頁預覽使用輕量 WebP
- 以深藍、金色、奶油色與五種角色專色重整首頁、題目、結果、說明及隱私區塊
- 新增標準大字與放大字體控制，放大模式下題目選項文字約為 21–23px，主要按鈕及選項具備更大的點選範圍
- 調整手機、平板與桌機版面，讓結果圖卡優先呈現，手機可分享至支援的社群 App，桌機可下載原始 PNG
- 新增自製圖卡風格的社群分享封面、燈塔網站圖示與自架繁體中文字型，避免不同裝置缺字
- 擴充離線快取、圖卡對應測試及靜態資產尺寸與格式驗證

### 驗證

- 七項自動化測試全數通過
- 全部 3,125 種作答組合仍維持唯一、可重現且五種角色各 625 種的平衡結果
- 320px 手機、390 × 844 手機、768px 平板、1440px 與 1920px 桌面瀏覽器 QA 通過
- 已驗證無水平溢位、主控台錯誤、外部請求或 localStorage 留存
- 五張正式 PNG 與使用者提供的原始檔案雜湊完全相同

## [1.0.0] - 2026-07-18

### 新增

- 完整的繁體中文職場韌力角色心理測驗
- 五題單題式自適應作答流程與工作階段續答
- 五種唯一結果及分層同分決勝演算法
- 手機原生圖卡分享與桌機 PNG 下載
- 1200 × 1500 Canvas 結果圖卡
- 鍵盤、螢幕閱讀器、深色模式與降低動態效果支援
- PWA、離線快取、SEO、社群預覽及 GitHub Pages 部署
- 3,125 種答案組合的窮舉測試與 GitHub Actions CI

### 驗證

- 每一種答案組合皆產生唯一且可重現的結果
- 五種角色各 625 種結果
- 桌機與手機瀏覽器 QA 全數通過
