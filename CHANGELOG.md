# Changelog

本專案的重要版本變更均記錄於此。

## [1.0.0] - 2026-07-18

### 新增

- 完整的繁體中文職場韌力角色心理測驗
- 五題單題式自適應作答流程與工作階段續答
- 五種唯一結果及分層同分決勝演算法
- 手機原生圖卡分享與桌機 PNG 下載
- 1200 × 1500 Canvas 結果圖卡
- 鍵盤、螢幕閱讀器、深色模式與降低動態效果支援
- PWA、離線快取、SEO、社群預覽及 GitHub Pages 部署
- 3,125 種答案組合的窮舉測試與 GitHub Actions CI

### 驗證

- 每一種答案組合皆產生唯一且可重現的結果
- 五種角色各 625 種結果
- 桌機與手機瀏覽器 QA 全數通過
