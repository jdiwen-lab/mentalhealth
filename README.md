# 職場韌力角色心理測驗

一個完全在瀏覽器端執行的繁體中文靜態測驗網站。受訪者會依序回答五個職場情境，系統以可重現的分層規則產出港灣型、燈塔型、海風型、珊瑚型或浪花型其中一種結果。

## 主要功能

- 響應式單題介面，支援手機、平板與桌面瀏覽器
- 選項順序隨機顯示，但以隱藏角色代碼計算，不受畫面位置影響
- 全部 3,125 種答案組合皆產出單一結果，五種角色在演算法上各占 20%
- 手機使用 Web Share API 分享 PNG 圖卡；桌面可下載 1200 × 1500 高畫質圖卡
- 作答與圖卡生成皆在瀏覽器內完成，沒有帳號、資料庫、追蹤碼或第三方分析
- 支援鍵盤操作、螢幕閱讀器、降低動態效果、高對比與系統深色模式
- 具備 PWA manifest、離線快取、SEO／社群預覽與 GitHub Pages 部署流程

## 本機執行

不需要安裝前端相依套件。Node.js 20 以上只用於測試與建置。

```bash
npm test
npm run validate
npm run build
npm run serve
```

接著開啟 `http://localhost:4173`。若要測試 `dist` 成品，可在 `dist` 目錄另行啟動靜態伺服器。

## 計分與同分決勝

1. 每題選項只記錄一種主要角色，各題地位相同。
2. 比較五題中各角色的出現次數；單一最高者直接成為結果。
3. 若同分，比較候選角色在第 1、2、5 題的出現次數。
4. 若仍同分，查看第 4 題的核心自我認定。
5. 若第 4 題角色不在候選名單，再依第 1、5、2、3 題順序找出第一個候選角色。

演算法實作位於 [`js/scoring.js`](js/scoring.js)，窮舉驗證位於 [`tests/scoring.test.mjs`](tests/scoring.test.mjs)。

## 部署

合併至 `main` 後，`Deploy GitHub Pages` 工作流程會驗證、建置並部署 `dist`。GitHub 儲存庫的 Pages Source 需設定為 **GitHub Actions**。

預設網址：<https://jdiwen-lab.github.io/mentalhealth/>

## 使用提醒

本測驗是職場韌力角色探索，結果只反映本次情境選擇中較常展現的應對方式，僅供自我覺察與活動體驗，並非心理衡鑑或醫療診斷工具。
